How Your CMS Works

1

Log in from anywhere

Go to yourdomain.com/admin.html from any browser — phone, tablet, or computer. Log in with your admin credentials.

2

Click Add New Event

Fill in the title, date, type, location, category, and description. You can also attach a photo and a report PDF link.

3

Upload a photo (optional)

Click the photo field and select an image from your device. It uploads directly to the server — no separate FTP needed.

4

Click Save

The event is saved to the database instantly. Your live website updates in real time — donors and visitors see it immediately.

5

Edit or Delete anytime

Use the pencil icon to edit an event or the trash icon to remove it. Changes are instant.

6

First-time setup

Run setup.sql once in Hostinger phpMyAdmin, then update db.php with your database name, username, and password from hPanel.

Tip: You can link to a PDF report hosted on Google Drive or Dropbox in the Report URL field. Donors can click to download the full report from your event card.
